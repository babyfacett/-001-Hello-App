# 01 Hello アプリ

このアプリはとてもシンプルな Streamlit のテンプレートです。アプリを実行すると、画面に「Hello, world!」のメッセージと簡単なインタラクションが表示されます。これをベースに新しいアプリを作成したり、Streamlit の基本的な構造を学んだりできます。

## 使い方

このフォルダーには `app.py` が含まれており、Streamlit でアプリを立ち上げることができます。

```bash
streamlit run app.py
```

このコマンドを実行すると、ブラウザでアプリが表示されます。タイトルと挨拶文に加えて、テキスト入力フィールドとボタンを用意しています。ボタンを押すと入力された内容がページに表示されます。
